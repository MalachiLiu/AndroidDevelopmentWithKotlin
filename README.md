# 作者序

很多人都听过费曼技巧，费曼技巧的核心是：**用自己的语言来记录或讲述你要学习的概念**。笔者也正是出于此进行此书的编写工作的，若是我的萤烛之光能够助人一二就更好了。

虽然在编写时已力求准确，但限于笔者水平，仍不免会有错误和疏漏。欢迎关于本书的任何反馈和提问。
